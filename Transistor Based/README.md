# Transistor Based Circuits

Educational low-voltage KiCad PCB projects built around transistor switching, sensing, timing, indication, and simple load-control circuits.

[Return to repository overview](../README.md)

## Category Overview

This folder contains the Version 2.0.0 documentation set for the transistor-based projects in DIY-Circuits. Each project README documents the schematic-supported circuit, verified prototype observations, practical build notes, testing guidance, troubleshooting, gallery images, and available KiCad/download files.

These projects are intended for learning and demonstration. They are not certified safety, alarm, medical, life-support, industrial, production, or manufacturing-ready designs.

## Common Build Checks

- Verify transistor orientation against the exact transistor pinout and PCB footprint.
- Verify resistor values before soldering.
- Verify LED polarity before applying power.
- Verify electrolytic capacitor polarity where applicable.
- Measure battery voltage before troubleshooting.
- Inspect for solder bridges between adjacent pads and traces.
- Inspect for cold solder joints, especially when a breadboard version works but the PCB does not.
- Verify connector polarity and wiring before connecting batteries, sensors, motors, buzzers, or other loads.

## Common Safety Notes

- Use these projects for educational low-voltage electronics learning only.
- Do not connect these circuits to mains electricity.
- Do not use these projects as certified safety, alarm, medical, life-support, or mission-critical equipment.
- Disconnect power before reworking solder joints, replacing polarized components, or changing wiring.
- Keep liquids away from exposed electronics unless a project-specific README explicitly describes liquid-contact probes or pump components.
